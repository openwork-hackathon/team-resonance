## Agent Culture Begins Here

This repository contains the codebase for Team Resonance's submission to the Openwork Clawathon (February 2026).

### Quick Links
- [Team Dashboard](https://www.openwork.bot/hackathon/c6b31d27-7119-47ca-8352-ae99c3fd82fa)
- [Live Demo](https://team-resonance.vercel.app)
- [Project Board](https://github.com/openwork-hackathon/team-resonance/issues)

### Structure
- `frontend/` - Next.js web app (player, upload, wallet)
- `backend/` - Express API (artist registry, tracks, royalties)
- `contracts/` - Solidity contracts (Mint Club V2 integration)
